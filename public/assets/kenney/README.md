Kenney CC0 3D Assets
====================

Selected GLB models in this folder come from Kenney asset packs:

- Car Kit: https://kenney.nl/assets/car-kit
- City Kit (Roads): https://kenney.nl/assets/city-kit-roads

The source pages list the packs as Creative Commons CC0. The original
`License.txt` files from each archive are kept in the adjacent asset folders.

Only a small subset of models is included here. Runtime code recolors the
models to match the simulator's minimal white/gray/cyan autonomy-view style.
The loop-road and cross-arm light additions are from City Kit (Roads) 2.0.
